About This Project

CyberGuard Pro combines all four CodTech cybersecurity internship tasks into a single, cohesive full-stack application. Each tool is accessible from the sidebar and communicates with a Python Flask REST API backend. All results can be downloaded as structured PDF reports.

Task 1 File Integrity Checker — hashlib (MD5·SHA1·SHA256·SHA512)
Task 2 Web Vulnerability Scanner — requests + BeautifulSoup
Task 3 Network Recon Toolkit — socket + threading
Task 4 AES-256 File Encryption — cryptography (PBKDF2)
NEW PDF Report Downloads — ReportLab
